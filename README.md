# Practice Projects

A collection of practice projects exploring different technologies and frameworks.

## Projects

- **prisma-practice** - Express server with Prisma ORM and PostgreSQL
- **Three JS** - Three.js practice projects including 3D scenes and camera controls

## Getting Started

Each project has its own setup instructions. Navigate to the project folder and check its README or package.json.

### Prerequisites

- Node.js (v18+)
- npm or yarn

### Running a Project

#### Prisma Practice
```bash
cd prisma-practice
npm install
npm run dev
```

#### Three JS Projects
```bash
cd "Three JS/<project-name>"
npm install
npm run dev
```

Available Three JS projects:
- `hello-world` - Basic Three.js scene setup
- `orbit-control` - Interactive 3D scene with orbit controls
- `orthographic-camera` - Orthographic camera implementation
- `solar-system` - Solar system visualization

## Technologies

- TypeScript
- Express
- Prisma ORM
- PostgreSQL
- Three.js
- Vite
- WebGL
