# Practice Projects

A collection of practice projects exploring different technologies and frameworks.

## Projects

- **prisma-practice** - Express server with Prisma ORM and PostgreSQL

## Getting Started

Each project has its own setup instructions. Navigate to the project folder and check its README or package.json.

### Prerequisites

- Node.js (v18+)
- npm or yarn

### Running a Project

```bash
cd prisma-practice
npm install
npm run dev
```

## Technologies

- TypeScript
- Express
- Prisma ORM
- PostgreSQL

## Notes

These are learning/practice projects for skill development.
